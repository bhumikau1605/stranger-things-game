# 🔦 Stranger Things — Crack the Upside Down Code

A spooky, atmospheric two-team code-cracking game built for live events and parties. Inspired by the world of Stranger Things.

---

## 🎮 How to Play

Each team gets their own page. A secret 4-letter code is generated using the letters **A, B, C, D** — shuffled every round.

- Look at the **your letters** tiles for the available letters
- Enter the correct **order** of the 4 letters
- Hit **Check Code** to see if you cracked it
- Wrong answer? The Upside Down laughs at you 👁️
- Crack the code to move to the next round!
